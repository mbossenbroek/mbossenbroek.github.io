get-epic-metrics
(get-epic-metrics scope ids metric-pattern)
Used to find the ids of epic style metric names.

get-metrics
(get-metrics env region metric)(get-metrics env region metric start end)
get-metrics*
(get-metrics* env region metric)(get-metrics* env region metric start end)
Retrieves one week of one minute data for a metric.

get-tag-list
(get-tag-list scope names)(get-tag-list tag scope names)(get-tag-list tag scope cq names)
Processes a list of tag requests in order. If the item is a string, it is
returned. If the item is a regex, the tags are retrieved and inserted into
the list. Any string items are excluded from the list returned by a regex.
get-tags
(get-tags scope name)(get-tags tag scope name)(get-tags tag scope cq name)
Gets tag names from Atlas. The default tag is 'name', meaning that it will
retrieve lists of metric names. Other tags are interpreted as dimensions of the
requested metric. Name can be a string, set, or regex.

upload-dashboard

Uploads a dashboard to atlas. This translates the queries to atlas's stack language first.
